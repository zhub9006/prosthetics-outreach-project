# Prosthetic Outreach Initiative

## Overview
Volunteer coordination project for our prosthetic outreach initiative, supporting task management and team collaboration across outreach teams in **Chicago, Boston, and Atlanta**.

## How to Use
1. **View Tasks**: Check the [Volunteer Task Board](tasks/) for current assignments.
2. **Claim a Task**: Edit `tasks/volunteer-tasks.md` to claim work.
3. **Report Progress**: Update task status and add comments.
